Projetos

BruteForce Automático – Python Pentest Tool

Este projeto consiste em uma ferramenta de BruteForce desenvolvida em Python com foco em testes de intrusão. Capaz de realizar tentativas automáticas de login utilizando listas customizadas de usuários e senhas, a aplicação simula ataques de força bruta para avaliar a robustez de sistemas de autenticação. Possui suporte para requisições HTTP personalizadas, detecção de respostas válidas, controle de tempo e exportação de resultados. Ideal para análises em ambientes controlados e auditorias ofensivas.

WebCrawler Inteligente – Coleta Avançada de Links

Ferramenta desenvolvida em Python para varredura e extração de links em sites com conteúdo estático e dinâmico (JavaScript), com suporte a BeautifulSoup, requests e Selenium. O WebCrawler identifica a estrutura de todos os hyperlinks válidos, respeitando profundidade de navegação, domínios permitidos e parâmetros dinâmicos. Pode ser usado em auditorias, análises de SEO, coleta de dados e automações de marketing digital.
